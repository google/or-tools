21 // An O(n^4) implementation of the Kuhn-Munkres algorithm (a.k.a. the
22 // Hungarian algorithm) for solving the assignment problem.
23 // The assignment problem takes a set of agents, a set of tasks and a
24 // cost associated with assigning each agent to each task and produces
25 // an optimal (i.e., least cost) assignment of agents to tasks.
26 // The code also enables computing a maximum assignment by changing the
27 // input matrix.
28 //
29 // This code is based on (read: translated from) the Java version
30 // (read: translated from) the Python version at
31 // http://www.clapper.org/software/python/munkres/.
