operations_research::MaxFlowStatusClass Class Reference

Detailed Description

We want an enum for the Status of a max flow run, and we want this enum to be scoped under GenericMaxFlow<>.

Unfortunately, swig doesn't handle templated enums very well, so we need a base, untemplated class to hold it.
