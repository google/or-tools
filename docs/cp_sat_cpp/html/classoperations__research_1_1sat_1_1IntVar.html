Detailed Description

An integer variable.

This class wraps an IntegerVariableProto. This can only be constructed via CpModelBuilder.NewIntVar().

Note
a BoolVar can be used in any place that accept an IntVar via an implicit cast. It will simply take the value 0 (when false) or 1 (when true).
