25// Validates that the input satisfies the following invariants:
26// 1. matrix.row_ids, matrix.column_ids, and matrix.coefficients are all the
27// same length.
28// 2. matrix.row_ids and matrix.column_ids are nonnegative.
29// 3. The matrix is in row major ordering with no repeats.
30// 4. Each entry in matrix.coefficients is finite and not NaN.
31// 5. If enforce_upper_triangular=true, then matrix must be upper triangular.

33 bool enforce_upper_triangular = false);
