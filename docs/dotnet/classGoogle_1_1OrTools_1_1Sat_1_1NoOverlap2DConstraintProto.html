NoOverlap2DConstraintProto

Detailed Description

The boxes defined by [start_x, end_x) * [start_y, end_y) cannot overlap.

TODO(user): Currently, this constraint will ignore a box if one of its dimension is zero. Add parameter to specify the behavior with boxes with zero area.
