# Shortest Path

The following variations are supported:

*   Dijkstra's algorithm for directed and undirected graphs with nonnegative
    weights: [here](shortest_path_dijkstra.md)
*   BFS for directed and undirected graphs with unit weights:
    [here](shortest_path_bfs.md)
*   Dynamic programming for shortest path in DAG (directed acyclic graph) with
    arbitrary weights: [here](shortest_path_dag.md)
*   Dynamic programming for shortest path in DAG with arbitrary weights and with
    multiple solutions: [here](multiple_shortest_paths_dag.md)
*   Constrained shortest paths in DAG with arbitrary weights and positive
    cumulative constraints; [here](constrained_shortest_path_dag.md)
